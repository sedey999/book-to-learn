# 变体模板示例

以下五种变体模板仅为**设计示例**，展示 book-to-learn 在"提示词与数据分离"架构下，同一套 items.json 数据可以通过修改提示词实现的不同学习场景。

> **重要说明**：这些变体并未在 skill 中预定义。skill 本身只提供四种内置推送模板（pdf-standard / pdf-large / feishu-card / feishu-card+image），变体玩法需要用户自行修改定时任务提示词来实现。每个模板的具体设计方案（配色、布局、字号、内容字段选择等）都可以灵活调整，没有固定格式。**原则上每本书定好模板后，后面每次推送都会按照同个模板样式生成。**

| 模板 | 文件 | 变体说明 | 核心改动 |
|------|------|----------|----------|
| 英文工具书·中英对照 | template-01 | 默认玩法 | 联网核对术语 + 中英对照翻译 |
| 英语单词·新闻结合 | template-02 | 单词学习变体 | 术语 → 联网搜索实时新闻 → 用术语讲解新闻 |
| 诗词海报·古典美学 | template-03 | 诗词学习变体 | 知识点 → 生成精美海报图片 |
| 复习模式·测验卡片 | template-04 | 复习模式变体 | 不推送新卡片 → 随机抽已推送卡片出题 |
| 深度模式·研究扩展 | template-05 | 深度模式变体 | 推送卡片 + 联网搜索论文/案例/实时数据 |

## 配色规范

所有模板示例遵循 book-to-learn 的统一配色体系：

| 颜色 | 色值 | 用途 |
|------|------|------|
| 绿 | #1a7f37 / #58cc02 | 核心观点、成功状态 |
| 蓝 | #0969da / #1cb0f6 | 解释、进度、链接 |
| 紫 | #8250df / #a560e8 | 金句、单词高亮 |
| 橙 | #bf8700 / #ff9600 | 应用场景、复习提示 |
| 红 | #cf222e / #ff4b4b | 术语、错误状态 |

## 使用建议

1. 拆书阶段照常执行，生成 items.json 数据
2. 根据学习目标，选择或设计一种变体提示词
3. 在 book_setup.py prompt 输出的提示词基础上修改
4. 确认每本书使用一种模板，定时任务会按该模板样式每日生成卡片
5. 如需切换模板，修改提示词即可，items.json 数据无需重新生成
